第一幕 魔术：双人亮魔杖

介绍 + 弟弟自我介绍

妈妈先起场，弟弟介绍自己；这一幕里妈妈和弟弟都要拿出自己的魔杖。

流程速记

  • 妈妈先开场，建立“霍格沃茨 / 小魔法师”氛围。
  • 弟弟接自我介绍，这是本幕最关键的长台词。
  • 妈妈记得 cue 弟弟拿自己的魔杖，不要只顾自己展示。
  • 两个人都亮魔杖后，再进入下一幕。

关键台词

弟弟长台词 大家好我叫高瑾博，是来霍格沃茨上学的小魔法师。

容易犯错

  • 妈妈自己拿完魔杖就往下走，忘了 cue 弟弟拿魔杖。
  • 弟弟长台词说太快，开场听不清名字和身份。
第二幕 魔术：盒子变领结和眼镜

看弟弟是不是收到了邀请

盒子里只变出领结和眼镜，没有介绍信。重点是展示动作慢、盒子始终正面朝前。

流程速记

  • 提前准备好盒子、领结、眼镜。
  • 妈妈拿出盒子给弟弟。
  • 弟弟先 放下魔杖，再拿起盒子给大家展示。
  • 展示要慢，而且 一定正面朝前。
  • 展示完放下盒子，再挥舞魔杖。
  • 之后从盒子里拿出领结和眼镜。
  • 推荐盒子 放在桌子上变，不要抱着。

关键提醒

动作重点 全程盒子都正面朝前，确保前面的门是关闭的。

容易犯错

  • 拿着盒子就直接挥魔杖，顺序乱掉。
  • 盒子侧过来或门没关好，前排容易看到机关。
  • 变的时候抱着盒子操作，画面不稳，也更容易穿帮。
第三幕 魔术：空牛皮纸袋变 3 个盒子

看弟弟是不是有魔法天赋

弟弟从空牛皮纸袋里变出 3 个盒子。重点是“空袋演示”一定慢，盒子正面也一定始终朝前。

流程速记

  • 提前准备好空牛皮纸袋和 3 个盒子。
  • 先演示袋子里没有东西，动作慢一些，让大家看清楚。
  • 弟弟开始变盒子时，妈妈负责串场，吸引大家注意力。
  • 拿出来的盒子正面朝前，放在桌子上也继续保持正面朝前。
  • 如果要收回去，也注意动作别穿帮。

关键提醒

动作重点 空袋子的演示慢一些，让大家看清楚没有东西。

容易犯错

  • 空袋展示太快，观众没建立“里面真的没有东西”的印象。
  • 盒子拿出来时角度歪，正面没有朝观众。
  • 变完收道具时太急，反而最容易露馅。
第四幕 魔术：空白书 → 线稿 → 彩色稿

看弟弟是不是能够在魔法学校好好学习

把空白页的书变成线稿，再变成彩色稿。重点是每次都先把书放桌上，再念咒语。

流程速记

  • 第一次展示空白页。
  • 变之前先把书 放在桌子上，念咒语 appear。
  • 翻出线稿页，翻的时候从 封皮后开始翻，而且要翻慢一点。
  • 第二次变之前还是先放桌上，再念咒语 color。
  • 最后翻出彩色稿。

关键台词 / 咒语

咒语 1 appear
咒语 2 color

容易犯错

  • 没把书先放桌上就直接变，动作看起来不稳。
  • 翻页太快，观众没看清空白页 / 线稿 / 彩色稿的层次。
  • 这里需要 多多练习，尤其是从封皮后开始翻这一点。
第五幕 魔术：魔杖变花

毕业前旁白 + 最后一个台上魔术

第五幕前先加过场旁白，再做“魔杖变花”。

关键旁白

过场旁白 转眼间，小魔法师已经在学校学习了 3 年，马上就要毕业了。

流程速记

  • 旁白结束后进入本幕。
  • 妈妈拿着花盆给弟弟。
  • 放花盆的动作要 慢慢放。
  • 抽的动作要 快速抽出来。

容易犯错

  • 放花盆和抽花盆速度反了，节奏会很怪。
  • 过场旁白说完没停顿就直接做魔术，情绪转场不清楚。
第六幕 魔术：袋子变礼物

长台词收束 + 最后送礼物

妈妈先问“最厉害的魔法是什么”，弟弟回答，再做最后一个袋子魔术。最后妈妈收口，两个人鞠躬。

流程速记

  • 妈妈先问弟弟：你这三年来，觉得最厉害的魔法是什么。
  • 弟弟说完长台词，再接“我还有最后一个魔法送给大家”。
  • 展示袋子的时候慢一点。
  • 袋子里礼物 少放几个，别塞太满。
  • 变完之后，妈妈补一句“结束之后发给大家礼物”，避免现场混乱。
  • 最后妈妈点题祝福，两个人鞠躬。

关键台词

妈妈提问 你这三年来，觉得最厉害的魔法是什么。
弟弟长台词 是每天都能跟小朋友们开心的玩耍和学习，是碰到了照顾我们的好老师们。
弟弟接魔术 我还有最后一个魔法送给大家。
妈妈收口 祝愿所有在座的小魔法师都能够在新的学校里好好学习，健康成长。

容易犯错

  • 袋子展示太快，观众看不清“里面没东西”。
  • 礼物放太多，操作容易乱，也更容易穿帮。
  • 变完不先说明“结束之后发礼物”，小朋友容易立刻往前冲。
  • 最后别忘了 两个人一起鞠躬。
排练建议：第二幕盒子、第三幕牛皮纸袋、第四幕翻书、第五幕花盆、第六幕礼物袋，都是高风险点，建议单独拆出来反复练。
